# Data folder info

The data files are too large to store in this repository.
You can download them here to populate the folder: [https://zenodo.org/records/14891299]

data_processing code used to convert all data into Huggingface-friendly format (ARROW-files): /multiauthor_data

data_tokenization used to flatten and tokenize said data to be ready for the task-specific adapter training: /mmbert_tokenized_dataset
