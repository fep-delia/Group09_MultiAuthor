# Data folder info

The raw data files are too large to store in this repository.
You can download them here to populate the folder: [https://zenodo.org/records/14891299]

The raw data has been preprocessed using data_processing.ipynb and is stored in a huggingface-friendly format (ARROW) in multiauthor_dataset 

The mmbert_tokenized_dataset contains this preprocessed data in tokenized format by mmBERT to allow for quicker setup for adapter training. Tokenization was done using data_tokenization.ipynb
